## Аукцион ненужных вещей.

Аукцион ненужных вещей - это место продажи лотов опережающих своё время. В пух и прах растерзанные критиками. Непринятые и непонятые. Когда-то они казались просто неудачными. Но, несмотря на критику, кажущуюся простоту или сложность, сейчас без этих лотов невозможно представить современную культуру. Когда в эти вещи не верил никто, продолжали верить их создатели. И сейчас эти лоты стали культовыми.

### Стек технологий

- CSS3

- Flexbox и Grid

### Запуск проекта

1. Склонируйте репозиторий с помощью команды:
   `git clone git@github.com:ibragim-ast/ono-tebe-nado.git`
2. Откройте проект в редакторе кода
3. Для запуска проекта убедитесь, что у вас установлено расширение (для VScode) Live Server, если его нет, загрузите его из магазина расширений.
4. Откройте index.html, в контекстном меню выберите пункт Open with Live Server, что автоматически откроет проект в дефолтном браузере.
